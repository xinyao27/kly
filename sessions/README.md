# Sessions

This folder contains session summaries for continuing work across conversations.

## File Format

`YYYY-MM-DD_HHmm_<topic>.md`

Example: `2024-12-24_1130_architecture-design.md`

## Usage

- **Save session**: Run `/save` at the end of a conversation
- **Resume session**: Start new conversation with "Read sessions/2024-12-24_1130_architecture-design.md and continue from there"

## Structure

Each session file contains:
- Summary of what was done
- Current project state
- Next steps (actionable items)
- Key files modified
